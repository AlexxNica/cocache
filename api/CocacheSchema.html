CocacheSchema Function

Validate a record against a schema.

This function is a NO-OP if no schema is specified.

Signature

CocacheSchema(record: Object, options: Object, displayName: String?)

Parameters (4)

record
Object
options
Object
options.schema
Object

An introspectable PropTypes schema like those provided by react-schema.

displayName
String?

Human-readable name of the cache that is storing this record. Used for logging purposes when there are errors.

    Exceptions

  • Error

    -

    If the structure of the record does not conform to the schema.
